# PracticePal 🎾

Tonight's tennis practice plan, sorted. Pick a coaching persona, set players / courts / minutes, and get a structured session plan — timeline, rotation rules and a competitive finisher — generated by Claude.

- **Live app:** https://luc1on.github.io/PracticePal/
- **Stack:** single `index.html` (vanilla JS), GitHub Pages, Supabase Edge Function (`practicepal` on the Command Centre project) holding the Anthropic API key.
- Sessions and smart defaults are saved in the browser (localStorage) — no accounts.

Built on the same pattern as the BasketBrain project.
